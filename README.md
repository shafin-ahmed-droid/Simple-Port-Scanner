# Simple Port Scanner 

A simple yet fast multithreaded port scanner built with Python.  
This project is designed for learning networking, sockets, and threading in Python.

---

## 📌 Features

- Fast multithreaded scanning
- Hostname to IP resolution
- Custom port range scanning
- Simple CLI interface
- Beginner-friendly codebase
